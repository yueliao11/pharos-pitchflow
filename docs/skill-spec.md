# Agent Narrated Deck-to-Video Skill — Specification

## One-liner

A reusable AI-Agent Skill that converts presentation decks, research reports, DAO updates, and on-chain analysis results into narrated videos with synchronized slides, voice-over, subtitles, cover image, and structured metadata.

## Why Agents need this

Most Pharos ecosystem Agents today return text, JSON, or raw transaction data. This Skill gives Agents a **native communication layer**: they can automatically explain their work in human-readable video, register a content hash on Pharos, and participate in the Agent Economy through payment, tipping, or gated distribution.

## Skill Input

`POST /skill/generate` accepts multipart/form-data:

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `pptx_file` | File | one of | A `.pptx` deck to convert. |
| `ppt_url` | string | one of | Remote URL of a `.pptx` deck. |
| `script_mode` | string | no | `auto_from_slide_text` (default) \| `speaker_notes` \| `custom_script` |
| `voice` | string | no | `female_en` (default) \| `male_en` \| `female_zh` \| `male_zh` \| `female_ja` \| `male_ja` \| `female_ko` \| `male_ko` |
| `language` | string | no | `en` (default) \| `zh` \| `ja` \| `ko` |
| `video_style` | string | no | `crypto_pitch` (default) \| `business` \| `education` \| `product_demo` |
| `output_format` | string | no | `mp4` (only supported currently) |
| `include_subtitles` | bool | no | Burn narration captions into the video. Default `true`. |
| `watermark` | bool | no | Add a small "Generated by PitchFlow Agent" watermark. Default `false`. |
| `callback_url` | string | no | Webhook called when generation finishes. |
| `custom_script` | string | no | Used when `script_mode=custom_script`. Split per slide by `---`. |
| `register_onchain` | bool | no | If `true` and Pharos env is configured, register content hash on-chain. |

## Optional On-chain Output

When `register_onchain=true` and the backend is configured with a Pharos RPC, Agent address and private key:

```json
{
  "pharos_tx_hash": "0x...",
  "content_hash": "0x...",
  "payment_required": false
}
```

The Skill writes the video `sha256` hash, title, video URI and metadata URI into `AgentVideoRegistry` on Pharos.

## Composability

This Skill is designed to be called by other Agents or Skills:

- **Wallet Analyzer Skill** → report PPT → **Deck-to-Video Skill** → wallet analysis video.
- **DAO Treasury Skill** → weekly data → **Deck-to-Video Skill** → DAO weekly video report.
- **Research Agent** → whitepaper summary → **Deck-to-Video Skill** → narrated pitch video.

Output metadata and content hashes let downstream Skills verify provenance, trigger payments, mint NFTs, or publish to social channels.
